Pengantar

Sistem bilangan adalah cara merepresentasikan kuantitas menggunakan simbol dan aturan penulisan. Yang paling umum adalah sistem bilangan basis-10 (desimal). Sistem bilangan berbeda-beda berdasarkan base (atau radix): banyak simbol yang dipakai untuk menyusun nilai.

  • Base (radix) = jumlah simbol unik (mis. desimal → 10 simbol: 0..9).
  • Posisi menentukan bobot: digit di sebelah kiri memiliki bobot lebih besar (basis^posisi).

Jenis-jenis Sistem Bilangan

1. Desimal (basis 10)

Simbol: 0,1,...,9. Digunakan sehari-hari.

2. Biner (basis 2)

Simbol: 0,1. Sering digunakan di komputer — setiap digit disebut bit.

3. Oktal (basis 8)

Simbol: 0..7. Kadang digunakan pada sistem lama atau representasi permission di UNIX.

4. Heksadesimal (basis 16)

Simbol: 0..9, A..F (A=10 .. F=15). Berguna untuk menulis nilai byte (mis: warna, alamat memori).

5. Bilangan Bulat, Rasional, Irasional, Real, Kompleks

Penjelasan singkat tipe bilangan: Integer (Z), Rational (Q), Irrational, Real (R), Complex (C).

Contoh & Konversi

Contoh: Konversi desimal → biner

Metode: bagi berulang dengan 2, catat sisa; baca sisa dari akhir ke awal.

// contoh: 13 (desimal) -> biner
13 ÷ 2 = 6 sisa 1
6 ÷ 2 = 3 sisa 0
3 ÷ 2 = 1 sisa 1
1 ÷ 2 = 0 sisa 1
Maka biner = 1101

Contoh: Desimal → Heksadesimal

// contoh: 254 (desimal) -> heksadesimal
254 ÷ 16 = 15 sisa 14 (E)
15 ÷ 16 = 0 sisa 15 (F)
Maka heksadesimal = FE

Contoh: Mengubah biner ke desimal

// contoh: 1011₂ -> ?₁₀
1011₂ = 1·2³ + 0·2² + 1·2¹ + 1·2⁰ = 8 + 0 + 2 + 1 = 11

Contoh tipe bilangan

  • Integer: ...,-2,-1,0,1,2,...
  • Rasional: 1/2, 3/4 (dapat ditulis sebagai pecahan)
  • Irasional: π, √2 (tak dapat ditulis sebagai pecahan sederhana)
  • Kompleks: 3 + 2i

Snippet: Script Konversi Sederhana

Berikut contoh fungsi kecil (Python & JavaScript) untuk konversi antar-basis.

# Python: desimal -> biner/hex

def to_base(n, base):
    if n == 0: return "0"
    digits = "0123456789ABCDEF"
    res = ""
    while n > 0:
        res = digits[n % base] + res
        n //= base
    return res

print(to_base(254,16))  # FE

// JavaScript: desimal -> biner/hex
function toBase(n, base) {
  return n.toString(base).toUpperCase();
}
console.log(toBase(13,2)); // 1101
